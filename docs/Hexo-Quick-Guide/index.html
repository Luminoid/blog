Quick Deployment

With Algolia and git deployment

$ hexo new "My New Article"
$ vim source/_posts/My-New-Article.md
$ hexo clean
$ hexo algolia
$ git add -A
$ git commit -m <msg>
$ git push

With Algolia and hexo deployment

$ hexo new "My New Article"
$ vim source/_posts/My-New-Article.md
$ hexo clean
$ hexo algolia
$ hexo deploy

Without Algolia

$ hexo new "My New Article"
$ vim source/_posts/My-New-Article.md
$ hexo deploy -g

Tag Plugins

Use tag plugins to reference images or other assets by relative paths instead of normal markdown syntax, otherwise these assets may be displayed incorrectly.

{% asset_img example.jpg This is an example image %}
{% asset_img "spaced asset.jpg" "spaced title" %}

Create

Initializes a website.

$ hexo init "My New Website"

Create a new article.

$ hexo new "My New Article"

Develop

Generate static files and watch file changes.

$ hexo generate -w

Start a local server.

$ hexo server

Index posts on Algolia. (This command runs without preceding hexo generate)

$ hexo algolia

Deploy

Deploys your website.

$ hexo deploy

Set the public_dir in _config.yml from public to docs if you want the GitHub Pages site built from the /docs folder in the master branch. More info here.

Building Hexo Website with Docker

Aug 22, 2017

This is a portable and generic solution to build Hexo website with Docker.

The Hexo base image

Creating Hexo Dockerfile

$ mkdir hexo
$ cd hexo
$ vim Dockerfile

Hexo Dockerfile Content

FROM ubuntu:14.04
MAINTAINER luminoid
ENV REFRESHED_AT 2017-08-28

# Install Hexo
RUN apt-get update -yqq \
  && apt-get install -yqq curl git-core \
  # Install Node.js Using a PPA
  && curl -sL https://deb.nodesource.com/setup_7.x | bash - \
  && apt-get install -yqq nodejs build-essential \
  # Set config to avoid permission issues
  && npm config set user 0 \
  && npm config set unsafe-perm true \
  && npm install -g hexo-cli

# Clone blog
WORKDIR /data
RUN git clone https://github.com/Luminoid/blog.git

# Install blog dependencies
WORKDIR /data/blog
RUN npm install \
  && npm install prompt --save \
  && npm install jquery --save

RUN mkdir -p /var/www/html/ \
  && cp -r /data/blog/docs/* /var/www/html/
VOLUME [ "/data/blog", "/var/www/html" ]
