Fibonacci Number in Different Programming Languages

Posted on 2022-02-22 Edited on 2022-02-27 In Programming Languages Symbols count in article: 2.9k Reading time ≈ 3 mins.

Programs for getting the nth Fibonacci number in different programming languages.

C (1972)

fibonacci.cview raw
#include <stdio.h>
#include <stdlib.h>

int fib(int n) {
    if(n <= 1) {
        return n;
    }

    int a = 0;
    int b = 1;

    for(int i = 2; i <= n; i++) {
        int c = a + b;
        a = b;
        b = c;
    }
    return b;
}

int main(int argc, char *argv[]) {
    int n = atoi(argv[1]);
    printf("fib(%d) = %d\n", n, fib(n));
    return 0;
}
$ gcc fibonacci.c
$ ./a.out 1
fib(1) = 1
