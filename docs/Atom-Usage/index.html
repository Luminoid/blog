Atom Usage

Posted on 2017-06-12 Edited on 2018-05-04 In Tool , Editor Views:

Official Site: https://atom.io

Keymaps

Universal Editor Keymaps

Packages

Tool

platformio-ide-terminal

A terminal package for Atom, complete with themes, API and more for PlatformIO IDE. Fork of terminal-plus.

script

Run code in Atom

linter

A Base Linter with Cow Powers

hydrogen

Run code and get results inline using Jupyter kernels like IPython, IJulia, and iTorch

atom-beautify

Beautify HTML, CSS, JavaScript, PHP, Python, Ruby, Java, C, C++, C#, Objective-C, CoffeeScript, TypeScript, Coldfusion, SQL, and more in Atom

split-diff

A split pane diff tool.

color-picker

Right click or press CMD-SHIFT-C/CTRL-ALT-C to open it.

Busy Signal

A package that provides an easy to use API to show your package is performing a task.

browser-plus

Real Browser in ATOM

UI

minimap

A preview of the full source code.

file-icons

Assign file extension icons and colours for improved visual grepping

tool-bar

Package providing customisable tool bar

highlight-selected

Highlights the current word selected when double clicking

pigments

A package to display colors in project and files.

qolor

An atom package to color your SQL queries!

minimap-highlight-selected

A minimap binding for the highlight-selected package

minimap-cursorline

Displays Atom cursorline in the minimap

minimap-selection

Display the buffer’s selections on the minimap

minimap-bookmarks

Displays Atom bookmarks in the minimap

minimap-find-and-replace

Minimap bindings for the find and replace package

minimap-pigments

An Atom plugin to display pigments colors in the Minimap.

minimap-autohide

Automatically hide the minimap when you aren’t using it.

Preview

preview

Ultimate previewer of source code in Atom.

markdown-preview-enhanced

The ‘BEST’ Markdown Preview you need for Atom Editor!

atom-html-preview

A live preview tool for Atom Editor.

pdf-view

Atom PDF viewer based on PDF.js

regex-railroad-diagram

Display railroad diagram of regex under cursor.

hex

View the hex dump of any file

IDE

atom-ide-ui

A collection of Atom UIs to support language services.

Nuclide

Nuclide is a collection of features for Atom to provide IDE-like functionality for a variety of programming languages and technologies.

Language

language-markdown

Adds grammar support for Markdown

language-babel

JavaScript ES201x, React JSX, Flow and GraphQL Grammar. Babel Transpiler

atom-typescript

The only TypeScript plugin you will ever need.

atom-ternjs

JavaScript code intelligence for atom with Tern. Adds support for ES5, ES6 (JavaScript 2015), Node.js, jQuery & Angular.

autocomplete-python

Python completions for packages, variables, methods, functions, with their arguments. Powered by your choice of Jedi or Kite.

autocomplete-clang

autocomplete for C/C++/ObjC using clang

linter-eslint

Lint JavaScript on the fly, using ESLint

docblockr

A helper package for writing documentation

Web

emmet

Emmet – the essential tool for web developers

atom-bootstrap4

Bootstrap4 Support with snippets and autocompletion.

Edit

sublime-style-column-selection

Enable Sublime style ‘Column Selection’. Just hold ‘alt’ while you select, or select using your middle mouse button. Also similar to Texmate’s ‘Multiple Carets’, or BBEdit’s ‘Block Select’

scroll-through-time

Two fingers scroll moves through time instead of space

Git

gitignore-snippets

Quickly create .gitignore files with templates from GitHub.

Coding

base64

A Base64 encoder and decoder

url-encode

URL encode or decode the selected text.
