## Key Bindings
{% post_link Universal-Editor-Keymaps %}

### Navigate
| Command | Key |
| ------- | --- |
| Open File in Assistant Editor             | `alt-click`       |
| Jump to Definition                        | `ctrl-cmd-j`      |
| Jump to Definition in Assistant Editor    | `ctrl-alt-cmd-j`  |
| Go Back                                   | `ctrl-cmd-left`   |
| Go Forward                                | `ctrl-cmd-right`  |

### View
| Command | Key |
| ------- | --- |
| Show Document Items | `ctrl-6` |
